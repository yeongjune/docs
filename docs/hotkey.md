# 开发工具快捷键集合


## mac版本的idea快捷键

```
打开属性管理【command+,】
全局搜索文件【control+H】
定位到方法的实现类【control+T】
全局搜索替换文件【shift+H】
重新编译改动过的文件 【command+R】
重新编译当前文件文件 【shift+command+F】
复制一行 【option+command+上下键】
快速移动一行【option+上下键】
显示当前类的方法名 【command+o】
生成get,set代码  【control+enter】
根据函数生成返回变量 【option+command+L】
查找类  【shift+command+R】
找出那些类调用了选中的方法 【control+option+H】
批量修改变量名称 【shift+F6】
把一段代码抽取为一个方法 【option+comman+M】
定位到文件的物理地址 ，鼠标放在编辑窗口中中的文件名，【command+鼠标单击】
导入包 【command+1】
修改类的方法名 【shift+F6】

常用的有fori/sout/psvm+Tab即可生成循环、System.out、main方法等boilerplate样板代码
例如要输入for(User user : users)只需输入user.for+Tab ；
再比如，要输入Date birthday = user.getBirthday()只需输入user.getBirthday().var+Tab即可。

快速修改变量名 【shift+F6】
//双斜杠注释  【command+/】
/**/代码块注释 【shift+control+/】
在代码里面输入 it  提示有循环trycatch等
写一个属性，调出生成提示  【option+command+J】
显示类的调用关系UML   【option+command+U】
优化导入的类和包  【Ctrl+Alt+O】
最近更改的代码   【CTRL+E】
打开文件目录   焦点定位到左侧目录树的节点上，然后使用【option+O】

```
